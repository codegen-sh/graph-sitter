# Graph-sitter Site

Minimal landing-page scaffold for the Vercel project that will own
`graph-sitter.com` after review.

## Local development

```bash
cd site
npm install
npm run dev
```

Open the printed localhost URL.

## Production build

```bash
cd site
npm run build
```

## Vercel setup

- Framework Preset: `Next.js`
- Root Directory: `site`
- Build Command: default
- Output Directory: default
- Install Command: default

Do not attach `graph-sitter.com` or `www.graph-sitter.com` until the docs domain
cutover is approved.
